=================
Insert a Document
=================

.. default-domain:: mongodb

method.

Expected Result
---------------

After you run the full example, you can find the following inserted
document in the ``restaurants`` collection:

.. code-block:: json
   :copyable: false

   {
     "_id": ObjectId("..."),
     "Name": "8282",
     "Cuisine": "Korean",
     "Borough": "Manhattan"
   }

For an example on how to find a document, see :ref:`golang-find-one`.
